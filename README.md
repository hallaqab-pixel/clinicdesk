## About
A clinic management system built with pure PHP supporting three roles: Admin, Doctor, and Patient.

---

## Setup
1. Copy project to `C:\xampp\htdocs\clinicdesk`
2. Import `clinicdesk_db.sql` in phpMyAdmin
3. Place AdminLTE 3.2 files inside `public/assets/adminlte/`
4. Open `http://localhost/clinicdesk`
